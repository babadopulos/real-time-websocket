# real-time-websocket
Java Web Application using WebSocket and Spring Boot

Build
-----
<pre>
$ mvn clean install
</pre>

Run from source
---------------
<pre>
$ export PORT=8080
$ mvn spring-boot:run
</pre>

Run from .jar file
------------------
<pre>
$ export PORT=8080
$ java -jar real-time-websocket-1.0-SNAPSHOT.jar
</pre>
